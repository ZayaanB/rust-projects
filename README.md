# rust-projects

**Current Project: Guessing Game**

## Cargo Commands:
| Command Name | Function |
|---|---|
| cargo new project_name | Creates a new project |
| cargo build | Creates a build with dependencies |
| cargo run | Runs the main project file |
| cargo help | Show cargo commands | 
| cargo check | Check codebase for errors without compiling | 

**Cargo:**
- Package manager for Rust
